Análise da questão
A questão é um exercício básico de programação que solicita a criação de um algoritmo cuja única função é exibir uma informação fixa na tela.

Os passos principais são:
1. Definir o nome que será exibido. Este valor não será lido do usuário.
2. Utilizar um comando de saída de dados para apresentar o nome definido na tela.
