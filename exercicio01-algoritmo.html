Exemplo para um possível algoritmo
Por ser uma tarefa de saída direta, este é o algoritmo mais simples possível.
Preparação Nenhuma preparação especial é necessária, pois o programa não receberá dados do usuário nem realizará cálculos.
Entrada Não há etapa de entrada de dados, pois o nome a ser exibido já está definido no próprio programa.
Processamento Não há etapa de processamento, pois a informação será exibida diretamente.
Saída 01 - Utilize um comando de impressão (como System.out.println()) para exibir o seu nome na tela. Ex: "Fulano de Tal".
