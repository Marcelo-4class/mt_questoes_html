Comandos para saída de dados
Os comandos de saída de dados em Java permitem exibir informações no console. Os principais métodos são System.out.println() e System.out.print(), que diferem no tratamento da quebra de linha.
Método Descrição Exemplos (código) Saída
System.out.println(); Exibe o texto ou valor passado como argumento e, ao final, realiza uma quebra de linha, posicionando o cursor na linha seguinte. System.out.println("Linguagem de programação"); Linguagem de programação
 
System.out.print(); Exibe o texto ou valor passado como argumento, mas não realiza a quebra de linha ao final. O cursor permanece na mesma linha após a impressão. System.out.print("Informe seu nome: "); Informe seu nome:
